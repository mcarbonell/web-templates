// 001 — Renderizado

Imágenes que se leen

Cada imagen se descompone píxel a píxel. El brillo de cada fragmento se traduce a un carácter ASCII, creando una representación visual hecha enteramente de texto.

Los caracteres más densos como @ # W representan zonas oscuras, mientras que los más ligeros como . : · representan la luz.

Tiempo real

Renderizado a 60fps usando Canvas API. Cada frame se procesa y convierte al instante sin lag perceptible.

Mapeo de densidad

27 niveles de caracteres ordenados por densidad visual para una representación fiel del brillo original.

Múltiples fuentes

Compatible con webcam, vídeo, imágenes estáticas y escenas 3D generadas proceduralmente.

Modo color

Cada carácter puede heredar el color del píxel original, creando ASCII art a todo color.
